Jordprøver QGIS Plugin

Version 1.0.2 — opdateret 24.04.2026

Installation i QGIS

  1. Åbn QGIS
  2. Gå til Plugins → Manage and Install Plugins → Settings
  3. Klik Add under "Plugin Repositories"
  4. Indsæt denne URL: 
    https://hrosenskjold.github.io/Jordprover/plugins.xml
  5. Gå til fanen All og søg efter Jordprover
  6. Klik Install Plugin
